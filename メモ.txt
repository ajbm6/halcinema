【諸注意】
・大きなレイアウト変更がない限り，テンプレにそって作成してください
・テンプレは記述ミスのある時以外なるべく改変しないでください
・タグの意味をきちんと理解して使うのがベターですが，
　参考にしたのが宇宙語で説明されていた場合，divとかそこらへんのに逃げてokです


●色はまだいいです(全体をつかめていない たぶん例の青っぽいページみたいになる)
●box-shadow gradiant とか細かなレイアウトもしなくていいです
●要素のサイズ感にはこだわってください（8の倍数がヒントらしいです しらんがな）



【ID,CLASS】
名前のつけかた：
    記号を入れない，2番目以降の単語は頭が大文字
    ◎…tagName
    ×…tag_name
    
    関連するものは名前を統一する
    #eigaTitle
    #eigaDate
    #eigaActor
クラスについて：
    仕様上どうしても！という時以外はなるべくidやセレクタで補う

反映優先順位：
    id > class > タグ名
    
【CSSかきかた】
    タグの書かれた順番でCSSを書くのがわかりやすいんじゃないかな
    優先度低いものはあとでもよい
    まとめられそうなところはまとめる
    なるべくブロック分けしてほしい
    
    あまり気にしなくていいが，まとまっていたほうがすっきりする
    /* ---------------------------------------------
        description
    --------------------------------------------- */
    xxx {
        display
        z-index
        position
        float
        top, right, bottom, left
        margin
        padding
        font
        color
        background
    }
